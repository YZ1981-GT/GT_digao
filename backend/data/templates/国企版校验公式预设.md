# 国企版 — 全量校验公式预设

> 本文档汇总国企报表附注中所有可预设的校验公式，涵盖：
> 1. 报表 vs 附注余额核对
> 2. 其中项（子项）校验
> 3. 同科目表间交叉校验
> 4. 跨科目交叉校验
> 5. 报表二级明细 vs 附注明细校验
>
> 核心原则：**本地规则引擎优先，LLM 兜底**

---

## 一、报表 vs 附注余额核对

### 1. 通用规则

每个科目下只有**第1个通过过滤的表格**参与报表 vs 附注余额核对。

过滤规则（以下类型的表格不参与余额核对）：
- 明细/分类子表（按账龄、按组合、坏账准备计提、已质押、已背书等）
- 资产组成部分子表（原价、累计折旧、减值准备单独列示的段落）
- 政策/描述性表格（折旧方法、计提政策等）
- 营业收入/成本的明细子表（按行业、按地区、按转让时间）

核对公式：`报表科目余额 = 附注第1个汇总表的合计行金额`

### 2. 特殊科目处理

| 科目 | 核对规则 |
|------|----------|
| 递延所得税资产 | 从"递延所得税资产和递延所得税负债"合并表中提取资产小计。若无小计行但"递延所得税资产"行本身有数据，则该行即为小计 |
| 递延所得税负债 | 从同一合并表中提取负债小计。若无小计行但"递延所得税负债"行本身有数据，则该行即为小计 |
| 其他应收款 | 附注中可能拆分为"应收利息"+"应收股利"+"其他应收款项"，取汇总表合计 |
| 其他应付款 | 附注中可能拆分为"应付利息"+"应付股利"+"其他应付款项"，取汇总表合计 |
| 营业收入/营业成本 | 附注中合并为一张表，分别提取收入合计和成本合计 |
| 固定资产 | 取多段表中"账面价值"段的期末余额合计 |
| 无形资产 | 同上 |
| 使用权资产 | 同上 |
| 投资性房地产 | 成本模式取账面价值段；公允价值模式取期末公允价值合计 |
| 商誉 | 账面原值期末 - 减值准备期末 = 账面价值 |

### 3. 合并口径 vs 母公司口径

- 合并口径：报表"合并"列余额 vs 合并附注中对应表格合计行
- 母公司口径：报表"公司"列余额 vs 母公司附注中对应表格合计行

母公司附注预设科目（国企版）：
1. 应收账款
2. 其他应收款
3. 长期股权投资
4. 营业收入和营业成本
5. 投资收益
6. 现金流量表补充资料

## 二、其中项（子项）校验

### 1. 通用规则

当表格中存在"其中："行时，其中项之和 ≤ 父项金额。

公式：`sum(其中子项) ≤ 父项金额`

> 注意：其中项之和可以小于父项（因为可能有未列示的其他子项），但不能大于父项。

### 2. 常见含"其中"项的科目

| 科目 | 表格 | 其中项说明 |
|------|------|-----------|
| 货币资金 | 货币资金分类 | 其中：受限制的货币资金 |
| 应收票据 | 应收票据分类 | 其中：已质押/已背书/已贴现 |
| 应收账款 | 按坏账准备计提方法分类 | 其中：单项计提/组合计提 |
| 其他应收款 | 按款项性质分类 | 其中：各性质子项 |
| 存货 | 存货分类 | 其中：各类存货子项 |
| 固定资产 | 固定资产情况 | 其中：各类别子项（房屋/机器/运输/电子等） |
| 在建工程 | 在建工程情况 | 其中：各工程项目 |
| 无形资产 | 无形资产情况 | 其中：各类别子项（土地使用权/专利权等） |
| 短期借款 | 短期借款分类 | 其中：质押/抵押/保证/信用 |
| 长期借款 | 长期借款分类 | 其中：质押/抵押/保证/信用 |
| 应付职工薪酬 | 应付职工薪酬列示 | 其中：短期薪酬/离职后福利/辞退福利 |
| 应交税费 | 应交税费明细 | 其中：增值税/所得税/城建税等 |
| 其他应付款 | 其他应付款项 | 其中：各性质子项 |
| 营业收入 | 按行业/产品/地区划分 | 其中：各分类子项 |
| 所得税费用 | 所得税费用明细 | 其中：当期所得税/递延所得税 |

## 三、同科目表间交叉校验

### 1. 坏账准备类科目

适用于：应收账款、其他应收款、应收票据、长期应收款

| 校验项 | 公式 |
|--------|------|
| 总表坏账准备 vs 坏账准备变动表 | `总表.坏账准备期末 = 坏账准备变动表.期末余额合计` |
| 总表金额 vs 分类表 | `总表.账面余额 = 单项计提表.小计 + 组合计提表.小计` |
| 总表坏账准备 vs 分类表 | `总表.坏账准备 = 单项计提表.坏账准备小计 + 组合计提表.坏账准备小计` |
| 坏账准备变动表勾稽 | `期初 + 计提 + 其他增加 - 收回或转回 - 核销 - 其他减少 = 期末`（宽表公式） |

### 2. 应付职工薪酬

| 校验项 | 公式 |
|--------|------|
| 汇总表 vs 短期薪酬明细表 | `汇总表."短期薪酬"行 = 短期薪酬明细表.合计行`（期初/增加/减少/期末各列） |
| 汇总表 vs 设定提存计划表 | `汇总表."离职后福利-设定提存计划"行 = 设定提存计划表.合计行` |
| 各表横向公式 | `期初 + 本期增加 - 本期减少 = 期末`（宽表公式） |

### 3. 固定资产

| 校验项 | 公式 |
|--------|------|
| 汇总表 vs 明细变动表 | `汇总表.账面价值期末合计 = 明细变动表.期末余额合计`（如有） |
| 多段纵向勾稽 | `账面价值 = 账面原值 - 累计折旧 - 减值准备` |
| 各段横向公式 | `期初 + 增加 - 减少 = 期末`（宽表公式，每段独立） |

### 4. 在建工程

| 校验项 | 公式 |
|--------|------|
| 汇总表 vs 项目变动表 | `汇总表.期末余额合计 = 项目变动表.期末余额合计` |
| 减值准备交叉 | `汇总表.减值准备期末 = 减值准备变动表.期末余额合计`（如有） |
| 项目变动表横向公式 | `期初 + 增加 - 转入固定资产 - 其他减少 = 期末`（宽表公式） |

### 5. 无形资产

| 校验项 | 公式 |
|--------|------|
| 汇总表 vs 明细变动表 | `汇总表.账面价值期末合计 = 明细变动表.期末余额合计`（如有） |
| 多段纵向勾稽 | `账面价值 = 原价 - 累计摊销 - 减值准备` |
| 各段横向公式 | `期初 + 增加 - 减少 = 期末`（宽表公式，每段独立） |

### 6. 使用权资产

同固定资产结构，多段纵向勾稽 + 各段横向公式。

### 7. 投资性房地产

| 校验项 | 公式 |
|--------|------|
| 成本模式多段纵向勾稽 | `账面价值 = 原价 - 累计折旧 - 减值准备` |
| 成本模式各段横向公式 | `期初 + 增加 - 减少 = 期末` |
| 公允价值模式横向公式 | `期初公允价值 + 购置 + 转入 + 公允价值变动 - 处置 - 转为自用 = 期末公允价值` |

### 8. 存货

| 校验项 | 公式 |
|--------|------|
| 分类表跌价准备 vs 跌价准备变动表 | `分类表.跌价准备期末合计 = 跌价准备变动表.期末余额合计` |
| 跌价准备变动表横向公式 | `期初 + 计提 + 其他增加 - 转回或转销 - 其他减少 = 期末`（宽表公式） |

### 9. 商誉

| 校验项 | 公式 |
|--------|------|
| 账面原值 vs 减值准备 | `账面原值表.期末合计 - 减值准备表.期末合计 = 报表商誉余额` |
| 账面原值表横向公式 | `期初 + 企业合并形成 - 处置 = 期末` |
| 减值准备表横向公式 | `期初 + 计提 - 处置 = 期末` |

### 10. 长期股权投资

| 校验项 | 公式 |
|--------|------|
| 分类表 vs 明细表 | `分类表.期末余额合计 = 明细表.期末账面价值合计` |
| 明细表横向公式 | `期初账面价值 + 追加 - 减少 + 投资损益 + 其他综合收益 + 其他权益变动 - 现金股利 - 计提减值 - 其他 = 期末账面价值`（宽表公式） |

### 11. 开发支出

| 校验项 | 公式 |
|--------|------|
| 横向公式 | `期初 + 内部开发支出 + 其他增加 - 确认为无形资产 - 计入当期损益 - 其他减少 = 期末`（宽表公式） |

### 12. 长期待摊费用

| 校验项 | 公式 |
|--------|------|
| 横向公式 | `期初 + 增加 - 摊销 - 其他减少 = 期末`（宽表公式） |

### 13. 债权投资 / 其他债权投资

| 校验项 | 公式 |
|--------|------|
| 总表减值准备 vs 减值准备变动表 | `总表.减值准备期末 = 减值准备变动表.期末余额合计` |
| 减值准备变动表横向公式 | `期初 + 增加 - 减少 = 期末`（宽表公式） |

### 14. 合同资产

| 校验项 | 公式 |
|--------|------|
| 总表减值准备 vs 减值准备变动表 | `总表.减值准备期末 = 减值准备变动表.期末余额合计` |

### 15. 递延收益

| 校验项 | 公式 |
|--------|------|
| 递延收益变动表横向公式 | `期初 + 增加 - 减少 = 期末`（宽表公式） |
| 政府补助表横向公式 | `期初 + 新增补助 - 计入损益 - 返还 + 其他变动 = 期末`（宽表公式） |
| 变动表合计 vs 政府补助表合计 | `递延收益变动表.期末合计 ≥ 政府补助表.期末合计`（政府补助是递延收益的子集） |

### 16. 应付债券

| 校验项 | 公式 |
|--------|------|
| 增减变动表横向公式 | `期初 + 本期发行 + 利息调整 - 本期偿还 = 期末`（宽表公式） |

### 17. 营业收入/营业成本

| 校验项 | 公式 |
|--------|------|
| 汇总表 vs 按行业划分 | `汇总表.收入合计 = 按行业划分表.收入合计` |
| 汇总表 vs 按地区划分 | `汇总表.收入合计 = 按地区划分表.收入合计` |
| 汇总表 vs 按转让时间划分 | `汇总表.收入合计 = 按转让时间划分表.收入合计` |
| 主营+其他=合计 | `汇总表.主营业务收入 + 汇总表.其他业务收入 = 汇总表.合计` |

## 四、跨科目交叉校验

### 1. 权益法投资损益三方核对

| 校验项 | 公式 |
|--------|------|
| 长期股权投资明细表 vs 利润表 | `长期股权投资明细表."权益法下确认的投资损益"合计 = 利润表.投资收益中"权益法核算的投资收益"` |
| 长期股权投资明细表 vs 投资收益附注 | `长期股权投资明细表."权益法下确认的投资损益"合计 = 投资收益附注."权益法核算的长期股权投资收益"行` |

### 2. 信用减值损失跨科目核对

| 校验项 | 公式 |
|--------|------|
| 信用减值损失附注 vs 各科目坏账准备变动 | `信用减值损失附注."应收账款坏账损失"行 ≈ 应收账款坏账准备变动表.(计提 - 收回或转回)` |
| 同理适用于 | 其他应收款、应收票据、长期应收款、债权投资、其他债权投资、合同资产 |

### 3. 资产减值损失跨科目核对

| 校验项 | 公式 |
|--------|------|
| 资产减值损失附注 vs 各科目减值准备变动 | `资产减值损失附注."存货跌价损失"行 ≈ 存货跌价准备变动表.(计提 - 转回或转销)` |
| 同理适用于 | 固定资产减值、在建工程减值、无形资产减值、商誉减值、长期股权投资减值 |

### 4. 固定资产 vs 在建工程

| 校验项 | 公式 |
|--------|------|
| 在建工程转固 vs 固定资产增加 | `在建工程项目变动表."转入固定资产"合计 ≈ 固定资产原值段."在建工程转入"合计`（如有该列） |

### 5. 开发支出 vs 无形资产

| 校验项 | 公式 |
|--------|------|
| 开发支出转无形资产 | `开发支出."确认为无形资产"合计 ≈ 无形资产原价段."内部研发"增加合计`（如有该列） |

### 6. 未分配利润勾稽

| 校验项 | 公式 |
|--------|------|
| 未分配利润变动 | `期初未分配利润 + 本期净利润 - 提取盈余公积 - 分配股利 ± 其他调整 = 期末未分配利润` |
| 本期净利润 vs 利润表 | `未分配利润附注."本期净利润"行 = 利润表."归属于母公司所有者的净利润"` |
| 提取盈余公积 vs 盈余公积附注 | `未分配利润附注."提取盈余公积"行 = 盈余公积附注."本期增加"中"提取"行` |

### 7. 现金流量表补充资料

| 校验项 | 公式 |
|--------|------|
| 净利润起点 | `补充资料."净利润"行 = 利润表."净利润"` |
| 固定资产折旧 | `补充资料."固定资产折旧"行 ≈ 固定资产附注."累计折旧"段本期增加合计` |
| 无形资产摊销 | `补充资料."无形资产摊销"行 ≈ 无形资产附注."累计摊销"段本期增加合计` |
| 长期待摊费用摊销 | `补充资料."长期待摊费用摊销"行 ≈ 长期待摊费用附注."本期摊销"合计` |
| 现金及现金等价物净增加额 | `补充资料."现金及现金等价物净增加额" = 现金流量表."现金及现金等价物净增加额"` |

### 8. 所得税费用调整

| 校验项 | 公式 |
|--------|------|
| 会计利润 × 税率 vs 所得税费用 | `所得税费用附注."会计利润" × 适用税率 + 各调整项 = 所得税费用` |
| 当期所得税 + 递延所得税 | `所得税费用明细."当期所得税费用" + "递延所得税费用" = 所得税费用合计` |

## 五、报表二级明细 vs 附注明细校验

### 1. 适用场景

部分报表科目在报表中有二级子明细行（如"其中：XX"），这些子明细金额应与附注中对应表格的明细数一致。

### 2. 常见二级明细科目

| 报表科目 | 报表二级明细 | 附注对应表格 | 校验公式 |
|----------|-------------|-------------|----------|
| 其他应收款 | 其中：应收利息 | 应收利息明细表 | `报表."应收利息"行 = 附注应收利息表.合计` |
| 其他应收款 | 其中：应收股利 | 应收股利明细表 | `报表."应收股利"行 = 附注应收股利表.合计` |
| 其他应付款 | 其中：应付利息 | 应付利息明细表 | `报表."应付利息"行 = 附注应付利息表.合计` |
| 其他应付款 | 其中：应付股利 | 应付股利明细表 | `报表."应付股利"行 = 附注应付股利表.合计` |
| 固定资产 | 其中：固定资产 | 固定资产情况表 | `报表."固定资产"行 = 附注固定资产表.账面价值期末合计` |
| 固定资产 | 其中：固定资产清理 | 固定资产清理表 | `报表."固定资产清理"行 = 附注固定资产清理表.合计` |
| 在建工程 | 其中：在建工程 | 在建工程情况表 | `报表."在建工程"行 = 附注在建工程表.期末合计` |
| 在建工程 | 其中：工程物资 | 工程物资表 | `报表."工程物资"行 = 附注工程物资表.合计`（如有） |
| 长期应付款 | 其中：长期应付款 | 长期应付款明细表 | `报表."长期应付款"行 = 附注长期应付款表.合计` |
| 长期应付款 | 其中：专项应付款 | 专项应付款表 | `报表."专项应付款"行 = 附注专项应付款表.期末合计` |
| 一年内到期的非流动负债 | 其中：一年内到期的长期借款 | — | `报表子项 = 附注对应表.合计`（如有单独表格） |
| 一年内到期的非流动负债 | 其中：一年内到期的应付债券 | — | 同上 |
| 一年内到期的非流动负债 | 其中：一年内到期的长期应付款 | — | 同上 |
