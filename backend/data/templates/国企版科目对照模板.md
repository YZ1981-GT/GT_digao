## 三、校验公式说明

### 报表 vs 附注余额核对

- 合并口径：报表"合并"列余额 vs 合并附注中第1个汇总表的合计行
- 母公司口径：报表"公司"列余额 vs 母公司附注中对应表格的合计行
- 每个科目下只有第1个通过过滤的表格参与余额核对

### 表间交叉校验（同科目内多表之间）

| 科目 | 校验规则 |
|------|----------|
| 应收账款/其他应收款/应收票据 | 总表坏账准备 = 坏账准备变动表期末合计；总表金额 = 单项+组合分类表合计 |
| 应付职工薪酬 | 汇总表"短期薪酬"行 = 短期薪酬明细表合计行 |
| 固定资产/无形资产/使用权资产 | 汇总表账面价值 = 明细变动表期末合计；纵向：账面价值 = 原值 - 折旧 - 减值 |
| 在建工程 | 汇总表期末 = 项目变动表期末合计；减值准备交叉核对 |
| 存货 | 分类表跌价准备 = 跌价准备变动表期末合计 |
| 商誉 | 原值表期末 - 减值准备表期末 = 报表商誉余额 |
| 长期股权投资 | 分类表期末合计 = 明细表期末账面价值合计 |
| 债权投资/其他债权投资 | 总表减值准备 = 减值准备变动表期末合计 |
| 合同资产 | 总表减值准备 = 减值准备变动表期末合计 |
| 营业收入/成本 | 汇总表合计 = 按行业/地区/转让时间划分表合计 |
| 递延收益 | 变动表期末合计 ≥ 政府补助表期末合计 |

### 跨科目交叉校验

| 校验项 | 公式 |
|--------|------|
| 权益法投资损益 | 长期股权投资明细表"投资损益"合计 = 投资收益附注"权益法核算"行 |
| 信用减值损失 | 信用减值损失附注各行 ≈ 对应科目坏账准备变动表(计提-收回) |
| 资产减值损失 | 资产减值损失附注各行 ≈ 对应科目减值准备变动表(计提-转回) |
| 在建工程转固 | 在建工程"转入固定资产"合计 ≈ 固定资产原值"在建工程转入"合计 |
| 未分配利润 | 期初 + 净利润 - 提取盈余公积 - 分配股利 ± 调整 = 期末 |
| 现金流量表补充资料 | 净利润 = 利润表净利润；折旧 ≈ 固定资产折旧段增加合计 |
